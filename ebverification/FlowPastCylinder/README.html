Flow Past a Cylinder

Case description

This case simulates 2D flow past a cylinder for Reynolds numbers \(\text{Re} = 10,500\). The geometry is defined as a rectangle with height \(D = 4\) [cm] and length \(L = 12\) [cm]. The inlet flow is constant in the positive x-direction with \(u(y) = U_{max}\), provided a Reynolds number

\[\text{Re} = \frac{\rho U_{max} D}{\mu},\]

where \(\rho\) is the density and \(\mu\) is the dynamic viscosity.
