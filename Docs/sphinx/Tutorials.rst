Tutorials
---------

The several cases in the *Exec/RegTests* directory are a good starting point for setting up a new PeleC case. In addition, there are several `tutorials from AMReX <https://amrex-codes.github.io/amrex/docs_html/Basics.html#example-heatequation-ex1-c>`_ that may be useful for general familiarity with how to build and run AMReX codes.
